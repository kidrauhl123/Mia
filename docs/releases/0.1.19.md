# Mia 0.1.19

- 修复窄窗口下浮空布局的返回、发送和中栏展开控制细节，消息列表切换到聊天页时更贴近桌面端窄屏交互。
- 调整主窗口浮空页面控制，减少中栏收起/展开后的视觉残留和错位。
- 本机运行时改为由 daemon 统一拥有，避免桌面进程重复接管 runtime 状态。
- 移动端会话列表改为产品级主列表：支持搜索、置顶、免打扰、标已读/未读，并用 Cloud 会话摘要计算预览和未读。
- Cloud settings 新增免打扰会话和手动未读覆盖字段，桌面、Web、移动端可通过同一 settings 事件同步。
- 移动端底部 Tab 增加消息/联系人角标，消息角标会排除免打扰会话；运行页状态文案改为中文并分组展示。
