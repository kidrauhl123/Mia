# Mia 0.1.24

- 新增 Mia Core 第一阶段拆分设计记录，为托管 Agent runtime、后台 daemon 和资源管理做结构准备。
- 增加 managed agent runtime 与通用进程启动器，统一桌面、daemon 和本地 Agent 进程的启动边界。
- 修复 Codex app-server 响应延迟，减少交互式 Codex 会话等待时间。
- 更新安装流程会在安装前停止后台服务，降低自动更新时文件被占用或旧进程继续运行的风险。
- 修复 Windows runtime 与 bot store 布局，并补充 Windows 标题栏相关主进程能力。
- 优化会话文件夹、胶囊标签、活动会话状态同步和 renderer 文本排版。
- 扩展本地 Agent、OpenClaw、Codex、daemon 和自动更新相关测试覆盖。
