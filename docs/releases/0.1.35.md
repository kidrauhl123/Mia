- 云端 Agent 运行时切换到 Claude Agent SDK 托管路径，并统一到 ACP session contract，减少旧 Hermes 会话状态残留。

- 修复技能市场开发态看不到内置技能的问题，技能市场会合并本地内置目录和云端缓存。
- 修复技能库滚动到下方后被分类栏重绘瞬间拉回顶部的问题。
- 更新 macOS 安装包和自动更新通道到 0.1.35。
