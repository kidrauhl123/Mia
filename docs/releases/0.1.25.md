# Mia 0.1.25

- Mia Core 成为打包后的 daemon 启动目标，并逐步取代旧的 Electron daemon 路径。
- 增加打包用 Node Core daemon、Mia Core helper app 组装和 launchd 目标解析，改善安装后后台服务启动一致性。
- 扩展 Core 侧 bot 执行能力，接入真实 Hermes 执行图、记忆、技能、附件和 MCP 上下文，并支持 cloud events / bridge invocation 路由。
- 将 engine model 解析、managed model 生命周期和四类 engine 运行路径收敛到 Mia Core，修复 OpenClaw fallback、显式 Core model 引用和 bridge runtime 引用传递。
- 完善 Core MCP 管理能力，包括 OAuth token 生命周期、连接诊断、外部 agent MCP 配置发现、secret 脱敏、禁用项和 tombstone 保留。
- 修复 bridge run 状态 streaming、停止委托、daemon shutdown 托管代理关闭，以及本机 engine model runtime 解析。
- 优化桌面侧搜索定位、搜索结果状态、会话输入草稿隔离、侧边栏滚动与官网下载交互。
- 更新移动端消息首页、我的页、设置排版、聊天入口和 SQLite chat cache。
