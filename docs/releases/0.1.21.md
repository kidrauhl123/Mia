# Mia 0.1.21

- 修复窄屏和重启后的聊天首屏定位，恢复会话和首次渲染后会保持在当前聊天内容区域。
- 优化桌面浮空聊天控件、通知和免打扰会话显示，消息界面在窄窗口下更稳定。
- 完善任务提醒链路：本地、云端和 Hermes 触发的提醒会走 Mia scheduler，并隐藏内部触发提示。
- 新增 scheduled task mode 与 schedule expression 共享 contract，Cloud release 会携带新的调度模式代码。
- 打通 Agent 权限审批到 daemon，Claude scheduler 工具会正确继承 bypass/permission 设置，避免恢复陈旧 Claude 会话。
- 同步移动端消息首页、搜索模式、Tab 角标和消息 chrome 优化。
- 优化响应式 shell、卡片网格、技能市场瀑布流刷新和能力/任务/助手库布局。
- 记录自定义 MCP 管理入口设计，并把 MCP 入口调整到能力库。
