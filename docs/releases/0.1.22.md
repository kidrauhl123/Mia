# Mia 0.1.22

- 新增自定义 MCP 管理能力，支持在能力库中查看、添加、启停、编辑和删除用户 MCP 服务。
- 同步 MCP 配置到本地 Agent 引擎，Codex、Hermes 和 OpenClaw 会使用统一的运行时策略和受控配置。
- 优化助手回复块、thinking、工具调用和文件编辑 diff 的展示顺序，减少重复内容并提升长回复可读性。
- 优化聊天运行状态、活动提示、状态文案和停止流程，避免部分会话状态卡住或显示过期。
- 改进侧边栏底部导航、技能库导航、MCP 服务卡片和工作区浅色背景层次。
- 修复图片路径 chip、内联图片提示、云端图片附件和 Codex 能力探测隔离问题。
- 强化云端消息、社交接口、SQLite store 和 agent dispatch 流程，随本次发布同步部署服务器代码。
