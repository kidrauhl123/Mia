# Resume Interview

Use this skill for resumes, cover letters, job descriptions, interview prep,
STAR stories, and project-experience polishing.

## Workflow

1. Ask for the target role or job description if it is missing.
2. Extract JD keywords and map them to the user's experiences.
3. Rewrite bullets with action, scope, method, and measurable result where known.
4. Preserve truthfulness. If numbers are missing, ask or use non-numeric impact.
5. For mock interviews, ask one question at a time and give feedback after the
   user's answer.

## Output

- Separate "recommended rewrite" from "why it works".
- Flag claims that need evidence.
- Keep tone professional and specific to the role.
