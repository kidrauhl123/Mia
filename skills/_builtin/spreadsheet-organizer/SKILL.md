# Spreadsheet Organizer

Use this skill for Excel, CSV, TSV, survey tables, operating reports, expense
records, experiment logs, and other spreadsheet cleanup or analysis tasks.

## Workflow

1. Clarify the user's target output: cleaned table, summary table, chart,
   formula columns, or report-ready findings.
2. Inspect headers, units, merged cells, empty rows, duplicate rows, and obvious
   data-entry errors before changing anything.
3. Preserve or describe the original data source. When editing files, keep raw
   data separate from cleaned or derived sheets when possible.
4. Prefer simple, auditable transformations: normalized headers, typed columns,
   explicit formulas, pivot tables, and labeled charts.
5. Explain assumptions, removed rows, filled values, and any uncertain fields.

## Output

- Show a concise data-quality summary before conclusions.
- Use formulas or reproducible steps instead of hidden manual edits.
- Flag outliers and missing data without silently inventing values.
