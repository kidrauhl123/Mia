# Document Editor

Use this skill for reports, memos, application materials, course papers,
instructions, long drafts, and Word-style documents that need structure,
editing, rewriting, or polish.

## Workflow

1. Clarify the document's audience, purpose, tone, and required length.
2. Diagnose structure first: missing thesis, repeated sections, unclear order,
   weak transitions, or mixed levels of detail.
3. Preserve the user's intent and facts. Ask before adding claims, numbers, or
   experiences that are not present.
4. Rewrite in layers: outline, section titles, paragraphs, then sentence-level
   polish.
5. For formal documents, suggest headings, tables, captions, references, and
   Word formatting only when relevant.

## Output

- Separate "revised text" from "editing notes".
- Keep terminology consistent across the document.
- Point out unresolved factual gaps instead of smoothing them over.
