# Lab Report

Use this skill for experimental data, CSV/XLSX tables, lab reports, charts,
basic statistics, and result interpretation.

## Workflow

1. Ask or infer what each column means, including units and experimental group.
2. Check missing values, obvious outliers, and whether rows are independent.
3. Choose simple analyses first: descriptive stats, trend, comparison, or
   correlation. Avoid advanced tests unless the user asks or the design needs it.
4. Explain every chart in terms of the experiment, not only the numbers.
5. Separate observation, statistical result, and interpretation.

## Output

- Include assumptions and data-cleaning decisions.
- Give report-ready wording only after the calculation or chart is clear.
- Do not overstate causality from correlation.
