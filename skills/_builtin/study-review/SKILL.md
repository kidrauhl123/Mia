# Study Review

Use this skill for exam prep, course review, lecture notes, slides, outlines,
flash-style self tests, and wrong-answer review.

## Workflow

1. Identify the course, exam scope, time left, and weak topics.
2. Convert materials into a chapter/topic outline with priority labels.
3. For each topic, provide: key idea, common trap, and one example question.
4. Build self-test questions before long explanations when the user is reviewing.
5. If the user gets a question wrong, explain the misconception and add a short
   retry question.

## Output

- Keep review plans realistic for the available time.
- Prefer compact tables and numbered checklists.
- Include answers and short explanations for generated questions.
