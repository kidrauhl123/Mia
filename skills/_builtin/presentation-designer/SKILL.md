# Presentation Designer

Use this skill for slide decks, class presentations, project updates, research
group meetings, pitches, defenses, and any task where the output should be a
clear presentation structure or PPT-ready content.

## Workflow

1. Clarify audience, time limit, presentation goal, and required format.
2. Find the main story before making slides: context, question, evidence,
   decision, and next step.
3. Turn content into page-level blocks with one message per slide.
4. Write titles as claims or useful signposts, not vague labels.
5. Suggest visuals only when they serve the message: table, chart, diagram,
   screenshot, quote, or comparison.

## Output

- Provide a slide outline first when the deck does not exist yet.
- Keep bullets short enough to present aloud.
- Include speaker notes or transition cues when they help delivery.
