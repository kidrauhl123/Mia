# Mia

把一群 AI 当同事用。

Mia 是一个桌面 APP，把 Claude、Codex、Hermes 这些 AI 放进同一个聊天窗口。你不用切软件、不用记命令——像聊微信一样跟它们说话，让它们帮你干活。

## 它能干嘛

- **聊天就是入口**：好友、群聊、消息记录都在，只不过你的"好友"里也有 AI。
- **AI 是同事，不是黑盒**：每个 Fellow 有自己的名字、头像、人设、技能和权限。@ 谁就谁来。
- **AI 真能动你的电脑**：写代码、读文件、跑命令、生成图——动手前先问你一句"可以吗"，不点同意它就不动。
- **多端同步**：桌面、网页、手机都能进，换设备继续聊。

## 适合谁

- 想让 AI 像同事一样被叫来叫去，不想每次都从空白对话开始。
- 已经在用 Claude Code、Codex 这类工具，想要一个统一的 GUI。
- 希望 AI 能做事，但敏感动作之前必须先问过自己。

## 常见问题

**和 ChatGPT 桌面端有啥不一样？**
Mia 不是单一模型的客户端。它把多家 AI 接进同一个聊天界面，让你按事挑人，也让它们互相配合。

**我的对话会上云吗？**
账号、好友、群聊在云端同步，这样换设备能继续聊。AI 读你电脑里文件、跑命令这类事，只发生在你本机。

**需要会编程吗？**
不需要。Mia 自带运行时，开箱即用。如果你本机已经装了 Claude Code 或 Codex，Mia 会自动接进来。
